# 🗺️ Система экспедиций

## Обзор

Экспедиции — исследование поверхности населённых и полностью разведанных пустых планет. Игрок отправляет часть экипажа на планету и исследует сетку 5×5 клеток, расходуя очки действий (AP).

Каждую планету можно исследовать **только один раз**.

---

## 🚀 Запуск экспедиции

1. Войдите на населённую планету или завершите разведку пустой планеты
2. Нажмите кнопку **«Исследовать поверхность»**
3. Выберите членов экипажа (уставшие недоступны)
4. Нажмите **«Запустить»**

Для экспедиции на пустой планете нужны завершённая поверхностная разведка и технология **«Комплекты экспедиции»**.

### Расчёт AP

```
AP = количество_участников + синтетики_в_отряде + технологические_бонусы
```

| Источник | Бонус |
|----------|-------|
| Каждый участник экипажа | +1 AP |
| Каждый синтетик в отряде | +1 AP дополнительно |
| Технология «Экспедиционные комплекты» (Tier 2) | +2 AP |

---

## 🗺️ Сетка 5×5

25 клеток со скрытым содержимым. Каждая клетка стоит **1 AP** для вскрытия.

### Типы клеток

| Иконка | Тип | Эффект |
|--------|-----|--------|
| 🏪 | Рынок | Кредиты на населённых планетах |
| 📦 | Тайник с грузом | Кредиты на пустых планетах |
| 🔬 | Лаборатория | Исследовательские ресурсы |
| 🏚️ | Руины | Событие с выбором |
| ⚠️ | Инцидент | Урон экипажу (20–35 HP) + потеря морали (−12) |
| ✨ | Артефакт | Редкий — артефакт (шанс мал) |
| 🚪 | Выход | Досрочное завершение |

### Распределение клеток по расам

Тип планеты влияет на вероятности тайлов. Например:
- **Люди** — больше рынков и лабораторий
- **Ксеносимбионты** — больше лабораторий и руин
- **Войдборн** — больше инцидентов и артефактов
- **Пустые планеты** — распределение зависит от точки интереса

### Точки интереса пустых планет

После завершения разведки пустая планета получает точку интереса. Она показывается перед запуском экспедиции и смещает вероятности клеток:

| Точка интереса | Основной акцент |
|----------------|-----------------|
| Древние руины | Руины и артефакты |
| Научный комплекс | Лаборатории и артефакты |
| Богатая ресурсная жила | Тайники с грузом |
| Место крушения | Тайники и опасные инциденты |
| Необычная биосфера | Лаборатории и инциденты |

На пустых планетах клетка рынка отображается как **тайник с грузом**; механика награды остаётся прежней.

---

## 🏚️ События в руинах

При вскрытии клетки «Руины» появляется модальное окно с описанием находки и **тремя вариантами** действий. Игрок должен выбрать один.

### Примеры событий

| Событие | Варианты |
|---------|----------|
| Тайник с припасами | Открыть / Сканировать / Уйти |
| Исследовательский архив | Скачать / Изучить дальше / Оставить |
| Сейф с кредитами | Вскрыть / Взломать / Пройти мимо |
| Старое оружие | Взять / Разобрать / Оставить |
| Чужое святилище | Осквернить / Изучить / Почтить |
| Зараженная зона | Обследовать / Дезактивировать / Отступить |
| Голографический архив | Скачать / Изучить / Пройти мимо |
| Капсула консервации | Вскрыть / Сканировать / Покинуть |

Каждый вариант имеет:
- **Безопасный исход** — гарантированная небольшая награда
- **Рискованный исход** — крупная награда или потеря
- **Отказ** — без последствий

---

## 💰 Награды

Накапливаются по ходу экспедиции и выдаются при завершении:

- Кредиты
- Торговые товары
- Исследовательские ресурсы (energy_samples, tech_salvage, rare_minerals, ancient_data, alien_biology)
- Артефакт (редко)

---

## ⚰️ Усталость экипажа

После завершения экспедиции все участники получают **5 ходов усталости**. Уставший экипаж:
- Не может участвовать в новых экспедициях
- Показывается серым с иконкой 😴 и счётчиком ходов

Усталость снимается автоматически: −1 за каждый ход.

---

## 📊 Итоги экспедиции

При завершении (кнопка «Завершить» или клетка «Выход»):

- Выдаются собранные награды
- Каждый участник получает **опыт**
- Все участники получают усталость (5 ходов)
- Мораль экипажа снижается (−10) — усталость от экспедиции
- Планета помечается как исследованная (кнопка исчезает)
- Ход автоматически продвигается

---

## 🔬 Технологии

| Технология | Тир | Эффект |
|-----------|-----|--------|
| Экспедиционные комплекты | 2 | Открывают экспедиции на полностью разведанных пустых планетах и дают +2 AP к каждой экспедиции |

**Требование для пустой планеты:** завершённая разведка поверхности.

---

## 🤖 Синтетики в экспедициях

Синтетики дают дополнительный AP сверх своей базовой единицы:
- 1 синтетик = +2 AP (1 как обычный член + 1 бонус)
- Бонус суммируется для каждого синтетика в отряде
