# 🧬 Сращивание ксеноморфов-симбионтов

## Обзор

Ксеноморфы-симбионты — уникальная раса, способная сращиваться с модулями корабля, предоставляя постоянные бонусы в зависимости от типа модуля.

## Как работает сращивание

1. **Требования:**
   - Член экипажа должен быть расы **ксеноморф-симбионт** (`xenosymbiont`)
   - Любая профессия может сращиваться
   - Ксеноморф должен находиться в модуле для сращивания

2. **Активация:**
   - Выберите ксеноморфа в панели экипажа
   - В списке заданий выберите **"🧬 Сращивание"**
   - Ксеноморф срастётся с текущим модулем

3. **Деактивация:**
   - Смените задание ксеноморфа на любое другое
   - Или переместите ксеноморфа в другой модуль
   - Бонусы сращивания будут потеряны

## Эффекты сращивания по модулям

| Модуль | Название эффекта | Бонусы |
|--------|-----------------|--------|
| **reactor** | Симбиоз с реактором | +10% выработка энергии |
| **cockpit** | Нейронная связь с мостиком | +5% уклонение корабля |
| **lifesupport** | Био-усиление систем | +20% кислород (вместимость экипажа), +2 HP регенерация экипажу за ход |
| **cargo** | Органическая упаковка | +10% вместимость груза |
| **fueltank** | Био-мембрана хранения | +15% эффективность топлива, +10% ёмкость топлива |
| **scanner** | Органическое сканирование | +20% дальность сканирования |
| **lab** | Био-лаборатория | +15% скорость исследований |
| **weaponbay** | Живое оружие | +10% урон оружия, +5% точность |
| **shield** | Щитовой симбиоз | +15% регенерация щитов, +10% ёмкость щита |
| **medical** | Регенеративная камера | +25% скорость лечения (только для тех, кто внутри модуля) |
| **engine** | Био-двигатель | +10% эффективность топлива |
| **drill** | Живой бур | +10% выход ресурсов при добыче |
| **ai_core** | Нейро-синтез | -50% шанс сбоя ИИ (для синтетиков) |
| **quarters** | Симбиоз с жилым модулем | +5 HP регенерация всему экипажу за ход |
| **repair_bay** | Симбиоз с ремонтным отсеком | +50% HP к ремонту дронами за ход |

## Важные замечания

- **Один ксеноморф = один модуль**: Каждый ксеноморф может срастись только с одним модулем
- **Автоматическое отсоединение**: При перемещении в другой модуль или смене задания сращивание сбрасывается
- **Накопление бонусов**: Несколько ксеноморфов могут срастись с разными модулями, бонусы суммируются
- **weaponShed не поддерживает сращивание**: Оружейные навесы не могут быть усилены

## Примеры использования

### Оптимальная стратегия

1. **Для исследования**: Срастите ксеноморфа с `lab` (+15% скорость) и `scanner` (+20% дальность)
2. **Для добычи**: Срастите с `drill` (+10% ресурсы) и `fueltank` (+15% эффективность)
3. **Для боя**: Срастите с `weaponbay` (+10% урон, +5% точность) и `shield` (+15% реген)
4. **Для выживания**: Срастите с `lifesupport` (+20% кислород) и `medical` (+25% лечение)
5. **Для экипажа**: Срастите с `quarters` (+5 HP реген всем) и `repair_bay` (+50% ремонт модулей)

### Комбинации бонусов

- **Топливная эффективность**: Ксеноморф в `engine` (+10%) + `fueltank` (+15%) = **25% экономия топлива**
- **Боевая мощь**: Ксеноморф в `weaponbay` даёт +10% урон и +5% точность одновременно
- **Защита**: Ксеноморф в `shield` увеличивает как регенерацию, так и максимальную ёмкость
- **Поддержка экипажа**: Ксеноморф в `quarters` (+5 HP/ход всем) + `medical` (+25% лечению внутри)

## Важные особенности отдельных модулей

- **medical**: бонус +25% лечения действует **только на экипаж внутри этого модуля**, а не на весь корабль
- **quarters**: бонус +5 HP/ход применяется **ко всему экипажу** глобально
- **repair_bay**: ксеноморф усиливает только **тот конкретный отсек**, с которым сращён
- **cargo**: при отмене сращивания возможен перегруз — груз не теряется, но до продажи излишков новые покупки заблокированы

## Технические детали

- Бонусы применяются автоматически каждый ход
- Бонусы к ёмкости (кислород, груз, топливо) пересчитываются динамически
- Бонусы к урону и точности работают как в бою, так и вне боя
- Бонус к уклонению влияет на шанс избежать атаки врага
