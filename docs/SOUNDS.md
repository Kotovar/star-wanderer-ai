# Звуковая Система

Звуки в игре синтезируются через Web Audio API и не требуют внешних файлов.

## Типы Звуков

| Звук | Описание | Когда используется |
|------|----------|-------------------|
| `click` | Короткий мягкий щелчок | UI клики |
| `success` | Восходящее арпеджио (C-E-G-C) | Успешные действия, наём экипажа |
| `error` | Нисходящий пилообразный тон | Ошибки, увольнение экипажа |
| `combat` | Агрессивный нисходящий паттерн | Начало боя |
| `travel` | Восходящая мелодия | Перемещение между секторами |
| `levelUp` | Длинное восходящее арпеджио | Повышение уровня экипажа |
| `shop` | Короткий торговый звук | Покупка оружия, товаров, модулей |
| `artifact` | Магическое звучание | Активация артефакта |
| `damage` | Резкий звук попадания | Попадание по врагу/игроку |
| `heal` | Мягкий исцеляющий звук | Лечение экипажа |
| `shield` | Защитный звук | Попадание по щитам |
| `energy` | Энергетический звук | Заправка топливом |
| `alert` | Тревожный сигнал | Засада, босс |
| `message` | Мягкое уведомление | Выполнение контракта, продажа товаров |
| `upgrade` | Технологичный звук | Улучшение модулей, ремонт |
| `destroy` | Разрушительный звук | Уничтожение модуля (утилизация) |

## Использование

```typescript
import { playSound } from "@/sounds";

// Базовое использование
playSound("success");

// С множителем громкости
playSound("alert", 1.5); // Громче
playSound("message", 0.5); // Тише
```

## Особенности

- **Вариативность**: Каждый звук имеет случайную вариацию высоты тона (±2%) для естественного звучания
- **Аудио контекст**: Используется единый AudioContext для всех звуков
- **Автоматическое восстановление**: Контекст автоматически возобновляется после приостановки
- **Без внешних файлов**: Все звуки синтезируются программно

## Добавление Нового Звука

1. Добавьте тип звука в `SoundType`
2. Добавьте конфигурацию в `soundConfigs`:

```typescript
mySound: {
    pattern: [440, 554.37, 659.25], // Частоты в Гц
    duration: 0.5, // Длительность в секундах
    waveType: "sine", // "sine" | "square" | "sawtooth" | "triangle"
    volume: 0.15, // Громкость 0-1
    delayBetweenNotes?: 0.1, // Задержка между нотами
}
```

3. Используйте `playSound("mySound")` в коде

## Типы Волн

- **sine**: Мягкие, приятные звуки (успех, лечение, артефакты)
- **square**: Резкие, 8-битные звуки (бой, тревога)
- **sawtooth**: Агрессивные звуки (ошибки, урон, разрушение)
- **triangle**: Нейтральные, технологичные звуки (улучшения, щиты)
