# 🪐 Уникальные фоны планет

## Обзор

Каждая планета теперь имеет уникальный визуальный стиль с:
1. **Анимированным CSS фоном** - градиенты и эффекты
2. **SVG визуализацией планеты** - красивое изображение на заднем плане

## Технические детали

### Файлы
- **Стили:** `/src/app/globals.css` (строки 5-528)
- **Утилита:** `/src/game/constants/planetBackgrounds.ts`
- **Компонент планеты:** `/src/game/components/PlanetVisual.tsx`
- **Компонент:** `/src/game/components/PlanetPanel.tsx`

### Структура SVG визуализации

Каждая планета состоит из:
1. **Градиенты** (`<defs>`) - основные цвета и свечение
2. **Основное тело** - круг с градиентной заливкой
3. **Детали поверхности** - уникальные для каждого типа (кратеры, дюны, полосы и т.д.)
4. **Атмосферное свечение** - тонкая обводка вокруг планеты

### Анимации

- **`float`** - плавное покачивание планеты (6 секунд)
- **Фоновые эффекты** - пульсация, вращение, мерцание

### Производительность

Все анимации используют CSS `transform` и `opacity`, которые оптимизированы для GPU:
- 60 FPS на большинстве устройств
- Минимальное влияние на CPU
- SVG рендерится один раз при монтировании

## Скриншоты

Для просмотра эффектов:
1. Запустите игру: `npm run dev`
2. Посетите любую планету на карте сектора
3. Обратите внимание на:
   - **Фон карточки** - анимированный градиент
   - **SVG планету справа** - покачивается и имеет уникальные детали

## 🎨 Расширение

### Добавление нового типа планеты

1. Добавьте CSS класс в `globals.css`:
```css
.planet-bg-новый-тип {
    background: linear-gradient(...);
    position: relative;
    overflow: hidden;
}
.planet-bg-новый-тип::before {
    content: "";
    /* анимация */
}
```

2. Добавьте маппинг в `planetBackgrounds.ts`:
```typescript
"Новый тип": "planet-bg-новый-тип",
```

3. Добавьте SVG визуализацию в `PlanetVisual.tsx`:
```typescript
"Новый тип": { primary: "#...", secondary: "#...", accent: "#..." }
```

4. Добавьте детали поверхности в SVG блок:
```typescript
{planetType === "Новый тип" && (
    <>
        {/* уникальные элементы */}
    </>
)}
```

## 📝 Примечания

- Фоны не влияют на читаемость текста благодаря полупрозрачному оверлею
- Анимации работают в фоновом режиме и не отвлекают от геймплея
- Все цвета соответствуют атмосфере типа планеты
- SVG планета позиционируется абсолютно и не влияет на поток документа
