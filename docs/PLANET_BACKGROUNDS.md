## Технические детали

### Файлы
- **Стили:** `/src/app/globals.css` (строки 5-380)
- **Утилита:** `/src/game/constants/planetBackgrounds.ts`
- **Компонент:** `/src/game/components/PlanetPanel.tsx`

### Использование

```typescript
import { getPlanetBackgroundClass } from "@/game/constants/planetBackgrounds";

const planetType = "Вулканическая";
const className = getPlanetBackgroundClass(planetType);
// Returns: "planet-bg-вулканическая"
```

### Структура CSS

Каждый фон состоит из:
1. **Базовый градиент** - основной цвет планеты
2. **Псевдоэлемент `::before`** - анимированный эффект
3. **Keyframe анимация** - определяет движение/пульсацию

### Производительность

Все анимации используют CSS `transform` и `opacity`, которые оптимизированы для GPU:
- 60 FPS на большинстве устройств
- Минимальное влияние на CPU
- Автоматическая пауза при скрытии панели

## Скриншоты

Для просмотра эффектов:
1. Запустите игру: `npm run dev`
2. Посетите любую планету на карте сектора
3. Обратите внимание на уникальный фон каждой планеты

## Расширение

Для добавления нового типа планеты:

1. Добавьте CSS класс в `globals.css`:
```css
.planet-bg-новый-тип {
    background: linear-gradient(...);
    position: relative;
    overflow: hidden;
}
.planet-bg-новый-тип::before {
    content: "";
    /* анимация */
}
```

2. Добавьте маппинг в `planetBackgrounds.ts`:
```typescript
"Новый тип": "planet-bg-новый-тип",
```

## Примечания

- Фоны не влияют на читаемость текста благодаря полупрозрачному оверлею
- Анимации работают в фоновом режиме и не отвлекают от геймплея
- Все цвета соответствуют атмосфере типа планеты
