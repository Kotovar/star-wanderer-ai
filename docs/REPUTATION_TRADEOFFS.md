# Дипломатические последствия

Изменение репутации с одной расой влияет на её союзников и соперников. Вторичное изменение рассчитывается по отношению расы:

`округление(основное изменение × отношение / 50)`

Например, выполнение расового контракта с наградой `+10` репутации даёт `-4` расе с отношением `-20` и `+3` расе с отношением `+15`.

Последствия показываются на карточке контракта до принятия и записываются в журнал после принятия. Доктрина не отменяет дипломатические последствия.
