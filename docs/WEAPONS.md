# Виды оружия (Weapons)

В этом документе описаны все типы оружия, их характеристики и боевые механики.

---

## 📋 Общая информация

Оружие устанавливается в модули типа `weaponbay` (Боевая палуба). В одну палубу можно установить несколько стволов. Каждый ствол стреляет **независимо** — у каждого выстрела свой отдельный бросок на точность.

**Критический удар** — один ролл на весь ход, применяется ко всем попавшим выстрелам одновременно.

---

## ⚙️ Общие механики

### Точность
- Базовая точность зависит от типа оружия
- Зажим: минимум **50%**, максимум **95%**
- Модификаторы: стрелок в палубе, ИИ-ядра, артефакт Ядро Прицеливания, задание `rapidfire` (штраф), задание `calibration` (бонус)

### Критический удар
- Базовый шанс: **5%**
- Базовый множитель: **×1.5**
- Артефакт **Критическая Матрица**: увеличивает шанс крита
- Артефакт **Матрица Перегрузки**: увеличивает множитель крита (+0.6 → итого ×2.1)

### Щиты
- Урон **всегда** идёт сначала по щитам, затем переполнение по модулям
- Исключение: Квантовая торпеда (полный обход щитов)

### Броня
- Броня (`defense`) модуля вычитается из урона по корпусу, минимальный урон = 1
- Кинетика игнорирует 50% брони
- Плазма перманентно разрушает броню

## 🔫 Типы оружия

### ◆ Лазерное (Laser)

| Параметр | Значение |
|---|---|
| Базовый урон | 20 |
| Точность | 95% |
| Бонус по щитам | ×1.2 |

**Механика:** Наносит усиленный урон щитам (+20%). Переполнение сверх щитов переходит на модуль по **базовому** урону (без бонуса). Лучший выбор против щитованных врагов на ранних этапах.

---

### ● Кинетическое (Kinetic)

| Параметр | Значение |
|---|---|
| Базовый урон | 15 |
| Точность | 90% |
| Пробитие брони | 50% |

**Механика:** Игнорирует 50% значения брони модуля при расчёте урона по корпусу. Эффективно против бронированных целей. Стандартный урон по щитам.

---

### ▲ Ракетное (Missile)

| Параметр | Значение |
|---|---|
| Базовый урон | 25 |
| Точность | 80% |
| Шанс перехвата | 20% |

**Механика:** Высокий урон, но низкая точность и риск перехвата. Шанс перехвата снижается при высоком модификаторе точности (от 5% до 50%). Стандартный урон по щитам и корпусу.

---

### ◉ Плазменное (Plasma)

| Параметр | Значение |
|---|---|
| Базовый урон | 30 |
| Точность | 85% |
| Бонус по щитам | ×1.3 |
| Разрушение брони | −10% за попадание |

**Механика:** Гибридное оружие. Бьёт сильнее по щитам (+30%). Каждое попадание **перманентно снижает броню** целевого модуля на 10% от исходного значения. После 10 попаданий броня модуля = 0, и всё последующее оружие наносит полный урон без вычета брони.

---

### ⬡ Боевые дроны (Drones)

| Параметр | Значение |
|---|---|
| Базовый урон | 18 |
| Точность | 85% |
| Выстрелов за ход | 1 |
| Стак урона | +5% за попадание (макс. +100%) |

**Механика:** Разгоняющееся оружие. Каждое попадание добавляет стак (+5% к урону дронов). Стаки сохраняются весь бой и сбрасываются при его окончании. При 20 стаках дроны наносят двойной урон. При промахе стак не растёт.

---

### ◈ Антиматерия (Antimatter)

| Параметр | Значение |
|---|---|
| Базовый урон | 40 |
| Точность | 78% |
| Бонус по щитам | ×2.5 |

**Механика:** Уничтожает щиты. Наносит в 2.5 раза больше урона по щитам. По корпусу (переполнение сверх щитов) — стандартный базовый урон. Лучший выбор для быстрого сноса щитов перед финальным ударом другим оружием.

---

### ◇ Квантовая торпеда (Quantum Torpedo)

| Параметр | Значение |
|---|---|
| Базовый урон | 55 |
| Точность | 62% |
| Обход щитов | 100% |

**Механика:** Полностью игнорирует щиты врага, урон идёт напрямую по целевому модулю. Низкая точность — компенсация за уникальную способность. Идеальна для уничтожения конкретного модуля независимо от состояния щитов.

## 💡 Тактические советы

- **Против щитов:** Антиматерия → Плазма → Лазер
- **Против брони:** Кинетика (быстрый снос), Плазма (постепенный)
- **Против конкретного модуля:** Квантовая торпеда
- **Затяжной бой:** Дроны (накапливают стаки и становятся сильнее)
- **Универсально:** Ракеты (высокий урон, но нужна поддержка точности)
