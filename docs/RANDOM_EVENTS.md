# Случайные события

Случайное событие может появиться после хода вне боя и путешествия. События не возникают до 6-го хода, после события действует перерыв 12 ходов, а последующая вероятность составляет 5% за ход. В среднем между событиями проходит около 25–35 ходов.

Пока капитан не примет решение, следующий ход недоступен. Выпавшие значения и цель события сохраняются, поэтому перезагрузка игры не меняет результат.

Каждое решение запускает одно отложенное последствие. Через 3 хода оно появляется в той же панели событий, применяет небольшой бонус или штраф, связанный с выбранным подходом, и записывается в журнал. Пока продолжение цепочки не проявилось, новое случайное событие не запускается. Состояние цепочки сохраняется; старые сохранения загружаются с пустой цепочкой.

## Выбор события (взвешенный)

События выбираются не равновероятно, а с учётом состояния корабля. Веса хранятся в `src/game/constants/randomEvents.ts` (`EVENT_REGISTRY`). Например: шторм чаще бьёт корабль без щитов; биологическая угроза чаще при отсутствии медика; дрейфующая капсула чаще встречается при наличии учёного; конфликт экипажа вспыхивает при низкой морали и большом составе.

## Структура выбора

Каждое событие предлагает 3 варианта:
- **Специалист** — требует конкретного члена экипажа (профессия), лучший результат + опыт.
- **Системы** — требует конкретный модуль, средний результат.
- **Стандарт** — доступен всегда, худший результат.

Чем сложнее требование — тем лучше исход.

### Бонус за специалиста

Выбор варианта «Специалист» дополнительно дарит временный эффект **«Решительная реакция»** (5 ходов): +4% уклонения и +1 реген здоровья в ход. Стимулирует держать под рукой нужные профессии и модули.

## Отложенные последствия

Через 3 хода после решения в той же панели появляется продолжение. Исход зависит от выбранного подхода («специалист» и «системы» считаются успешными, «стандарт» — нет).

| Событие | Успешный подход | Стандарт |
|---------|-----------------|----------|
| Шторм / Вирус / Метеориты | +мораль (2–3) | −мораль (2) |
| Биологическая угроза | +3 мораль | −3 мораль |
| Конфликт экипажа | +3 мораль | −2 мораль |
| Сигнал бедствия | +3 мораль | −2 мораль |
| Утечка топлива | +4 топливо | −3 топливо |
| Дрейфующая капсула | данные Древних / тех. лом | +10₢ |
| Пиратское нападение | +20₢ | −10₢ |
| Торговец | тех. лом / +15₢ | — |
| Брошенный корабль | данные Древних / тех. лом | +10₢ |
| Древний сигнал / Научный прорыв / Резонанс артефакта | данные Древних / тех. лом | +1 мораль (резонанс) / — |

## ПЛОХИЕ события (лучший исход — ничего не потерять)

### Космический шторм
- Пилот в кабине снижает урон и получает опыт.
- Щиты поглощают удар.
- Стандарт: модуль получает полный урон.

### Метеоритный дождь
- Инженер уклоняется, снижая урон, и получает опыт.
- Щиты поглощают поток.
- Стандарт: модуль получает полный урон.

### Бортовой вирус
- Медик уменьшает потерю морали и получает опыт.
- Ядро ИИ или синтетик устраняет вирус ценой 5 HP.
- Стандарт: весь экипаж теряет мораль.

### Утечка топлива
- Инженер уменьшает потерю топлива и получает опыт.
- Ремонтные дроны сохраняют топливо ценой 5 HP ремонтного отсека.
- Стандарт: полная расчётная потеря топлива.

### Биологическая угроза
- Медик снижает урон экипажу и получает опыт.
- Жизнеобеспечение нейтрализует споры ценой 5 HP.
- Стандарт: экипаж получает урон и теряет мораль.

### Пиратское нападение
- Стрелок контратакует, пираны бросают добычу: +кредиты.
- Оружейный отсек отпугивает пиратов.
- Стандарт: потеря кредитов, мораль -3.

## ХОРОШИЕ события (лучший исход — наибольшая награда)

### Дрейфующая капсула
- Учёный извлекает данные Древних и получает опыт.
- Инженер с грузовым отсеком разбирает капсулу: +2 тех. лома.
- Стандарт: продажа координат за кредиты.

### Сигнал бедствия
- Медик спасает экипаж: +40₢, мораль +5.
- Мед-отсек принимает раненых: мораль +3.
- Стандарт: игнорирование, мораль -3.

### Странствующий торговец
- Учёный находит данные Древних в товарах.
- Грузовой отсек: закупка припасов, +топливо, +кредиты.
- Стандарт: ничего.

### Брошенный корабль
- Учёный: полный анализ, +данные Древних, +тех. лом.
- Сканер: глубокое сканирование, +2 тех. лома, +кредиты.
- Стандарт: быстрый осмотр, небольшая награда.

### Древний сигнал
- Учёный в лаборатории: полная расшифровка, +2 данные Древних, +тех. лом.
- Сканер: частичный перехват, +1 тех. лом.
- Стандарт: игнорировать, ничего.

### Научный прорыв
- Учёный в лаборатории подтверждает гипотезу: +2 данные Древних, +2 тех. лома и опыт.
- Сканер собирает проверочные данные: +1 тех. лом.
- Стандарт: только запись в журнале.

### Резонанс артефакта
- Учёный настраивает резонанс: +2 данные Древних, мораль +5 и опыт.
- Лаборатория выделяет фрагменты: +1 данные Древних, +1 тех. лом.
- Стандарт: наблюдение, мораль +1.

## НЕЙТРАЛЬНЫЕ события

### Конфликт экипажа
- Пилот-капитан уменьшает потерю морали и получает опыт.
- Жилой модуль: развод смен без потери морали.
- Стандарт: полная потеря морали.

Недоступные решения отображаются в интерфейсе вместе с их требованиями. Старые сохранения загружаются без активного события.
