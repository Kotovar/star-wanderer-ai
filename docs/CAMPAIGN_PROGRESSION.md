# Прогрессия кампании

Перед перелётом в сектор игра сравнивает боевую готовность корабля с рекомендуемым значением тира. Оценка учитывает исправные активные модули, их уровень, вооружение и уровни живого экипажа.

Рекомендуемые значения: тир 1 — 12, тир 2 — 22, тир 3 — 34, тир 4 — 48. Если готовность ниже 80% рекомендации, карта показывает предупреждение, но не запрещает перелёт.

## Обычные противники

Базовая угроза обычного противника равна тиру сектора. При генерации она может отклониться на один уровень; чёрная дыра добавляет ещё один уровень. Итоговый диапазон — 1–6. Угроза увеличивает количество модулей, урон, щиты, кредитную награду и исследовательские ресурсы через существующую боевую систему.

Уже созданные враги в сохранениях сохраняют прежний уровень угрозы. Новая кривая применяется при создании новой галактики.

## Экономика боя

Команда `npm run check:economy` сравнивает среднюю кредитную награду обычного боя с ожидаемыми расходами на топливо, ремонт и лечение для тиров 1–4. Проверка использует реальные игровые цены и средний коэффициент боевой награды.

Средняя победа должна полностью оплачивать восстановление, а чистый доход должен расти вместе с тиром. Крупные покупки модулей, экипажа и дипломатии остаются основными долгосрочными расходами. Симулятор намеренно не учитывает торговлю, контракты и исследовательскую добычу, поэтому проверяет консервативный сценарий.

Целевой коридор обязательных расходов после среднего боя — 15–50% кредитной награды. Для достижения нижней границы цена топлива установлена в 3₢ за единицу; стоимость ремонта и лечения не изменялась.

Если на станции топлива не хватает для самого дешёвого доступного прыжка и кредитов недостаточно для покупки дефицита, станция бесплатно выдаёт ровно недостающее количество. Каждая станция помогает только один раз за игру; использованные станции сохраняются в сохранении.

## Источники дохода

`npm run check:income` сравнивает средние денежные результаты боя, типового контракта, разведки и выгодного торгового рейса. Контракт остаётся доходнее одиночного боя, поскольку требует нескольких действий; торговля даёт меньшую безопасную прибыль; разведка занимает промежуточное положение.

Денежная награда разведки масштабируется на 50% за каждый тир сектора: базовый диапазон 100–250₢ сохраняется на первом тире, а поздняя разведка не обесценивается относительно остальных активностей.
